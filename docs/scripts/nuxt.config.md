# Nuxt configuration file

Nuxt can be configured easily with one single file, called `nuxt.config`, with either a `.js`, `.ts` or `.mjs` extension.

```ts
import { defineNuxtConfig } from 'nuxt3'

export default defineNuxtConfig({
  // My Nuxt config
})
```
